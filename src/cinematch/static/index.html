Film recommendations, explained

CineMatch scores similarity with TF–IDF over catalog text—ideal for demos, interviews, and iterating toward richer models. Switch between a compact demo CSV and a local MovieLens extract without changing code.

  • Content-based
  • Pydantic + OpenAPI
  • Production-shaped layout

Run a query

Choose a data source, enter a title, and set how many neighbors to return.

Demo uses data/sample_movies.csv (or CINEMATCH_DATA_CSV). MovieLens reads movies.csv + ratings.csv from the path on the server.

Quick tips Demo sample titles include Arrival, Paddington 2. Broader titles (e.g. Jumanji) need MovieLens with CINEMATCH_MOVIELENS_DIR. Fixture: Toy Story.
